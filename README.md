
# Recipes Index 
## website: http://benkolber.app


Hello & welcome to the Recipes Index - Get your desired recipes easily and smartly from a remote api.<br>
The website showcase the use of the following technical skills:
* React.js
* Node.js
* Redux
* JavaScript
* Web API
* CSS
* HTML
